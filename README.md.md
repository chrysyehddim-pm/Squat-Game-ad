# 🏆 金幣深蹲王 (Squat Game PoC)

這是一個結合 Google MediaPipe AI 姿態辨識與遊戲化機制的數位健康互動 MVP。
旨在透過超級瑪莉風格的互動，鼓勵長輩進行深蹲運動，並於背景紀錄臨床醫療數據（關節角度、配速）至 Firebase。

## ✨ 核心功能
* **AI 姿態辨識**：不需穿戴裝置，透過網頁鏡頭即時捕捉人體骨架。
* **醫療級雙關節引擎**：嚴格判定膝蓋與髖關節彎曲角度，防止危險代償姿勢。
* **智慧語音引導**：具備防重疊機制的語音提示，適合長輩遠距離互動。
* **雲端數據追蹤**：遊戲結束自動結算最大/最小彎曲角度與配速，並推上 Firebase。

## 🚀 如何在本地端測試？
1. 下載此專案資料夾。
2. 使用 VS Code 或 Cursor 打開資料夾。
3. 安裝 **Live Server** 擴充套件。
4. 點擊 `index.html`，點擊右鍵選擇 `Open with Live Server`。
*(注意：基於瀏覽器隱私安全，視訊鏡頭必須在 localhost 或 HTTPS 環境下才能啟動)*
